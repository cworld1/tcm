# TCM

An ai program.

## Installnation

Install scaffold:

```bash
pip install --upgrade poetry
```

Install independencies:

```bash
poetry install
```

## Run the code

```bash
poetry run python ./main.py
```

- Main color: `#066335`
- Background color: `#BFF1D8`
- Border color: `#44B17B`
